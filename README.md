# othello_AI

## Prikazy

### Start
START \<farba bota> <čas ktorý má bot na odpoveď>

### Move
MOVE <zakódovaný stav hry>

String 64 znakov, každé políčko je jeden z nasledujúcich znakov.
'-' - prázdne políčko
'O' - biely coin
'X' - čierny coin

Na toto program odpovie najlepším nájdeným ťahom pre svoju farbu.

### Stop
STOP

Ukončí program.
